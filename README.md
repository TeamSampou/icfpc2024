## Usage

* Install [the Haskell Tool Stack](https://docs.haskellstack.org/en/stable/)
* Place your token (e.g. `????????-????-????-????-????????????`) in `token.txt`.
* `stack repl`
* Show scoreboard, etc.
  * `> cat "scoreboard"`
* Submit solution
  * `> submitSolution "lambdaman1" "LLLDURRRUDRRURR"`
